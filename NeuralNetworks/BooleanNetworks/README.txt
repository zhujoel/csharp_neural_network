This folder contains three serialized networks:
 - initial.json contains the serialization of a network with one hidden layer; an input of size 2 and an output of size 1. The weights of this network were randomly initialized.
 - and-trained.json contains the serialization of a network obtained from initial.json after training it to recognize the AND boolean function.
 - xor-trained.json contains the serialization of a network obtained from initial.json after training it to recognize the XOR boolean function.
